Para implementar IP sobre ATM, simplemente en cada enrutador se maneja una lista de posibles interfaces de salida, según el destino al que se dirige el paquete IP que se reciba. Mientras se está iniciando una conexión entre dos dispositivos en la red, el siguiente paso en la ruta se escoge aleatoriamente de la lista y se registra temporalmente en una base de datos, hasta que transcurra cierto tiempo y no se reciban más paquetes correspondientes a la misma conexión. La información a registrar puede ser IP origen, IP destino, así como números de puerto origen y destino.
